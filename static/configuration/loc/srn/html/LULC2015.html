National Land Use Land Cover Map 2015

This map shows the national Land Use Land Cpver of 2015.

SBB, NIMOS and other relevant key partners created this map in close collaboration. The process to create the map involves:

  • Work sessions with the key partners where the classes are defined,
  • Field validation and
  • Image interpretation and classification using ancillary data
