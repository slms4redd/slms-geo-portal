Beschermde Gebieden en LBB-reserves

Deze laag geeft een overzicht van de beschermde gebieden in Suriname. Deze file is geproduceerd op basis van staatsbesluiten en informatie van LBB. Volgende types werden geidentificeerd:

  • Natuurreservaten
  • Multiple Purpose Management Areas (MUMA)
  • Speciaal beschermd bos
  • Natuurpark
  • LBB-reserves
  • Voorgestelde natuurreservaten
